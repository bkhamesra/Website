Classifying Black Hole Signatures with Supervised Machine Learning

In the following notebooks, we analyze the gravitational waves data obtained from numerical simualations with help of machine learning. Gravitational waves are perturbations on the fabric of space-time created by mergers of massive objects like black holes and neutron stars. These waves inherit the information of the original source configuration and hence, can be used to learn more about objects millions of lights years away from Earth. In this work, we compare various Supervised ML method to predict the souce configuration based on the gravitational wave signatures emitted. The github repo contains necessary codes and jupyter notebooks and datasets used for this work (click the above link)
